# Day 004 Glossary

| Term | Simple Meaning | Beginner Example |
| --- | --- | --- |
| Database | A system that stores data used by applications. | An app stores customer login details in a database. |
| Operational Database | A database used for daily business operations. | An orders database used by an online shopping website. |
| OLTP | Online Transaction Processing; systems designed for day-to-day transactions. | Saving a new payment transaction quickly. |
| Data Warehouse | A system that stores clean, structured data for reports and analytics. | A monthly sales table used by a dashboard. |
| Analytical System | A system designed to answer business questions using data. | A system calculating total revenue by country. |
| OLAP | Online Analytical Processing; systems designed for analysis and large queries. | Calculating yearly sales by product category. |
| Data Lake | A storage area for raw, mixed, or large data in many formats. | A place storing CSV files, JSON logs, PDFs, and images. |
| Object Storage | Storage for files or objects such as documents, logs, images, and videos. | A cloud bucket storing product images. |
| Raw Data | Data kept close to its original form. | A raw CSV export from a marketing platform. |
| Clean Data | Data that has been fixed and standardized. | Dates converted to one format and duplicates removed. |
| Business-Ready Data | Data prepared for reports, dashboards, and decisions. | A daily sales summary table. |
| Table | Data organized in rows and columns. | An orders table with order ID, date, and amount. |
| File | A stored item such as CSV, JSON, PDF, image, or video. | A PDF invoice or JSON export file. |
| Storage Layer | A stage or area where data is stored during its journey. | Raw layer, clean layer, or warehouse layer. |
| Ingestion | Bringing data from a source into a data platform. | Loading order data from a database into a data lake. |
| Transformation | Changing data into a cleaner or more useful format. | Creating monthly revenue from daily order records. |
| Reporting | Presenting data in reports for business use. | A weekly sales report. |
| Dashboard | A visual report with charts, metrics, and trends. | A Power BI dashboard showing sales and orders. |
| BigQuery | A cloud data warehouse from Google. | A company stores reporting tables in BigQuery. |
| Azure Synapse | A Microsoft analytics and data warehouse platform. | A team uses Azure Synapse for large reporting queries. |
| Azure Data Lake | Microsoft cloud storage for data lake use cases. | A company stores raw JSON and CSV files in Azure Data Lake. |
| Google Cloud Storage | Google cloud object storage for files and data. | A company stores raw exports and images in Google Cloud Storage. |
| Snowflake | A cloud data warehouse platform. | Analysts query clean sales tables in Snowflake. |
| Amazon S3 | Amazon cloud object storage commonly used for data lakes. | Raw logs and files are stored in Amazon S3. |
