# Day 001 Glossary

| Term | Simple Meaning | Beginner Example |
| --- | --- | --- |
| Data | Facts, records, or information about something. | A customer's order amount, order date, and product name. |
| Raw Data | Data in its original form before cleaning or organizing. | Customer name stored as `" sai "` with extra spaces. |
| Clean Data | Data that is corrected, standardized, and ready to use. | Customer name stored as `"Sai"` and email stored in lowercase. |
| Data Source | The place where data comes from. | A website, mobile app, database, payment system, or Excel file. |
| Data Pipeline | A process that moves and prepares data from source to destination. | Moving daily order data from a website to a reporting database. |
| Data Ingestion | Collecting data from a source and bringing it into a data system. | Loading customer orders from an app into storage every day. |
| Data Storage | Saving data in a place where it can be managed and used later. | Storing order records in a database or data warehouse. |
| Data Transformation | Changing data into a cleaner or more useful format. | Converting `"1000"` from text into the number `1000`. |
| Data Quality | The level of correctness, completeness, and trustworthiness of data. | Checking that every order has an order date and amount. |
| Database | A system used to store and manage structured data. | A table that stores customer details and order details. |
| Data Warehouse | A system used to store organized data for reporting and analysis. | A sales reporting system that stores cleaned order data. |
| Data Lake | A storage area that can hold large amounts of raw or processed data. | A place where a company stores files, logs, and transaction data. |
| Dashboard | A visual report that shows important information using charts and numbers. | A sales dashboard showing total revenue and top products. |
| SQL | A language used to read and work with data in tables. | Asking a database for total sales by day. |
| Python | A programming language often used to automate and process data tasks. | Reading a CSV file and cleaning customer names. |
| Cloud | Online computing services used to store and process data on external platforms. | Storing company files in cloud storage instead of one local computer. |
| ETL | Extract, Transform, Load: collect data, clean/change it, then store it. | Clean order data first, then load it into a reporting table. |
| ELT | Extract, Load, Transform: collect data, store it first, then clean/change it. | Load raw order data into storage, then transform it later for reports. |
