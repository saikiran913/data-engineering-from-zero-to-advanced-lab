# Day 001 Practice Questions

## Section 1: Simple Questions

1. What is data?
2. What is Data Engineering in simple words?
3. Why is raw data not always useful?
4. What does a data engineer do with messy data?
5. What is a data pipeline?
6. Why do companies need clean data?
7. What is the difference between a data engineer and a data analyst?
8. Why is SQL important in Data Engineering?
9. Why is Python useful for data engineers?
10. What can happen if a business report is created from incorrect data?

## Section 2: Scenario-Based Questions

1. An online store wants to know total sales for yesterday, but order data is stored in three different systems. What problem does the company have?
2. A bank sees some transactions with missing transaction times. Why can this be a problem?
3. A food delivery app stores city names as `Hyderabad`, `hyd`, and `HYDERABAD`. Why should this data be cleaned?
4. A marketing team runs ads on different platforms. Why might they need a data engineer before creating a final report?
5. A streaming platform wants to know which shows are watched the most. What type of data might it collect?

## Section 3: Think Like a Data Engineer

1. If customer names have extra spaces before and after them, how would you clean them?
2. If order amounts are stored as text, why might you convert them into numbers?
3. If the same order appears twice in a dataset, what issue can it create in a sales report?
4. If data arrives late every day, how can it affect dashboards?
5. If you are given data from a website, mobile app, and payment system, how would you think about organizing it?

## Section 4: Mini Assignment

Pick one real-world app you use daily. It can be an online shopping app, food delivery app, banking app, streaming app, fitness app, or learning app.

Write short answers for the following:

- What data does the app collect?
- Where might the data be stored?
- How might the company use that data?
- What problems can happen if the data is incorrect?

Try to think like a data engineer. Focus on how data is collected, cleaned, stored, and used.

## Answer Key

1. Data means facts, records, or information.
2. Data Engineering means building systems that collect, store, clean, transform, and deliver data for use.
3. Raw data may be messy, incomplete, duplicated, or incorrectly formatted.
4. A data engineer cleans, organizes, transforms, and prepares messy data.
5. A data pipeline is a process that moves data from one place to another and prepares it for use.
6. Companies need clean data so reports and decisions are accurate.
7. A data engineer prepares and moves data, while a data analyst uses data for reports and insights.
8. SQL is important because it helps work with data stored in tables.
9. Python is useful for automation, file handling, and data processing.
10. Incorrect data can create wrong reports, which can lead to wrong business decisions.
